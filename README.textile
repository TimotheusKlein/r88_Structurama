	<p><strong>Allgemeines:</strong><br />
Das Programm arbeitet zur Erzeugung einer synthetischen Bevölkerung mit aggregierten Strukturdaten, die in einer Shape- bzw. der dazu gehörenden DBF – Datei abgelegt sind (z.B. Einwohner), in Verbindung mit statistischen Angaben zur Verteilung beliebiger Attribute (z.B. Altersgruppen). Dabei wird weitestgehend auf einen simplen, mehr oder weniger gelenkten Monte-Carlo-Algorithmus zurückgegriffen. Die Einwohner der erzeugten synthetischen Bevölkerung können in einem weiteren Schritt unter Berücksichtigung ihrer Merkmale zu Gruppen wie z.B. Haushalten zusammengefasst werden. Abschließend können auch diesen Gruppen Merkmale zugewiesen werden.<br />
Das Programm wurde mit python 2.5.4. geschrieben.<br />
Weitere Informationen im Download.</p>

	<p><strong>General Information:</strong><br />
In order to create synthetic populations, the program uses aggregate data that is provided in shape- oder dbf-files (e.g. inhabitants), combined with statistical information on the distribution of arbitrary attributes (e.g. age groups). This process uses a rather simple, more or less guided Monte-Carlo-Algorithm. The inhabitants of a synthetic population may be grouped according to their attributes to form e.g. households. Finally these groups may have further attributes assigned to them.<br />
The program was written on python 2.5.4.<br />
More information in the download.</p>
